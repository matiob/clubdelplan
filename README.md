### Features

- Search and list plans.
- View the details of plans.
- Create users.
- _Login_ and _logout_ users.
- JWT to validate API requests
- Password management
- Login with Google
- Mobile-first
- Advanced search and filters
- Voting and user comments
- CRUD of plans
- User session persistence
